Code related to reproduce various analyses in "Genomic evolution and natural history of myeloproliferative neoplasms on therapy".

## Signature analysis on the discovery dataset

The code and supporting input data files for this discovery analysis are in the markdown bulk_signature_analysis/20260410_bulk_HDP.Rmd 

## Tree based validation signature analysis 

The code and supporting input data files for this discovery analysis are in the markdown in phyloanalysis/mpn_tn_scc_signatures_validation.Rmd.  The output markdown html for the results reported in the supplementary materials is in html/mpn_tn_scc_signatures_validation.html and additionally a subsequent run with more iterations is presented in html/mpn_tn_scc_signatures_validation_1000_iterations.html.

Data availability:

Manifests for samples deposited in EGA are located in "data/EGA_manifests".

See Mendeley ( doi: 10.17632/hsz7nkwxgy.1 ) for VCFs of the called somatic variants for the phylogenetic trees of the newly reported donors PD63423, PD54831, PD54832 and PD6577
